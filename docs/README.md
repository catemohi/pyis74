# Документация pyis74

Этот каталог содержит публичную документацию библиотеки.

- [Публичные методы](public-api.md) - клиенты, домены API, входные параметры,
  возвращаемые модели и безопасные примеры данных.

Все примеры в документации синтетические. Не переносите в публичные документы реальные
адреса, номера договоров, `USER_ID`, `BUILDING_ID`, MAC-адреса, UUID камер,
подписанные media URL и access token.
