# Документация pyis74

Этот каталог содержит публичную документацию библиотеки.

## Ссылки пакета

- [PyPI package](https://pypi.org/project/pyis74/) - опубликованный пакет `pyis74`.
- [GitHub repository](https://github.com/catemohi/pyis74) - исходный код и issue tracker.
- [GitHub Releases](https://github.com/catemohi/pyis74/releases) - релизные артефакты,
  changelog и provenance/attestation-файлы.

## Содержание

- [Публичные методы](public-api.md) - клиенты, домены API, входные параметры,
  возвращаемые модели и безопасные примеры данных.
- [Changelog](../CHANGELOG.md) - изменения между релизами и текущий unreleased scope.

Все примеры в документации синтетические. Не переносите в публичные документы реальные
адреса, номера договоров, `USER_ID`, `BUILDING_ID`, MAC-адреса, UUID камер,
подписанные media URL и access token.
